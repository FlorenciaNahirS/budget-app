## How to install

 1. `git clone https://github.com/FlorenciaNahirS/budget-app`
 2. `cd budget-app`
 3. `cd client `
 4. `npm i`
 5. `cd ..`
 6. `npm i`
 7.  Create a `.evn` file
 8. Copy the content of `.env.example` file and add your credentials
 9. `sequelize db:generate`
 10. `sequelize db:migrate` 
 11. `sequelize db:seed:all`
 12. `npm start`
